# Error Messages

## Post

| 메시지 | 발생 상황 |
|---|---|
| `해당 게시물이 존재하지 않습니다.` | 게시글 삭제, 수정, 수정 권한 검증, 단건 조회 시 대상 게시글을 찾지 못한 경우 |
| `게시물이 존재하지 않습니다.` | 조회수 증가 대상 게시글을 찾지 못한 경우 |
| `해당 회원이 존재하지 않습니다.` | 게시글 작성/삭제 시 로그인 회원을 찾지 못하거나 게시글 조회 메서드에서 대상 게시글을 찾지 못한 경우 |
| `삭제 권한이 없습니다.` | 회원 게시글을 비로그인 사용자나 다른 회원이 삭제하려는 경우 |
| `비밀번호가 올바르지 않습니다.` | 비회원 게시글 삭제 시 비밀번호가 없거나 일치하지 않는 경우 |
| `제목은 필수입니다.` | 게시글 작성/수정 시 제목이 비어 있는 경우 |
| `내용은 필수입니다.` | 게시글 작성/수정 시 태그 제거 후 실제 내용이 없는 경우 |
| `작성자는 필수입니다.` | 비회원 게시글 작성/수정 시 작성자명이 비어 있는 경우 |
| `비밀번호는 필수입니다.` | 비회원 게시글 작성 시 비밀번호가 비어 있는 경우 |
| `해당 게시물에 대한 수정 권한이 존재하지 않습니다.` | 게시글 수정 페이지 접근 또는 수정 요청 시 작성자가 아닌 사용자가 접근한 경우 |
| `비밀번호가 일치하지 않습니다.` | 비회원 게시글 수정 페이지 접근 시 비밀번호가 없거나 일치하지 않는 경우 |
| `검색어를 입력해주세요.` | 게시글 검색 시 검색어가 `null`인 경우 |
| `검색어를 2글자 이상 입력해주세요.` | 게시글 검색 시 검색어가 2글자 미만인 경우 |

## Comment

| 메시지 | 발생 상황 |
|---|---|
| `해당 게시물이 존재하지 않습니다.` | 댓글 작성 시 대상 게시글을 찾지 못한 경우 |
| `내용은 필수입니다.` | 댓글 작성 시 댓글 내용이 비어 있는 경우 |
| `해당 회원이 존재하지 않습니다.` | 회원 댓글 작성 시 로그인 회원을 찾지 못한 경우 |
| `작성자는 필수입니다.` | 비회원 댓글 작성 시 작성자명이 비어 있는 경우 |
| `비밀번호는 필수입니다.` | 비회원 댓글 작성 시 비밀번호가 비어 있는 경우 |
| `{commentId}번 댓글이 존재하지 않습니다.` | 댓글 삭제 시 대상 댓글을 찾지 못한 경우 |
| `postId랑 commentId가 일치하지 않습니다. postId={postId}commentId={commentId}` | 댓글 삭제 요청의 게시글 ID와 실제 댓글의 게시글 ID가 다른 경우 |
| `삭제 권한이 없습니다.` | 댓글 삭제 시 로그인 회원 정보를 찾지 못한 경우 |
| `해당 댓글을 삭제하지 못했습니다.` | 댓글 작성자, 관리자, 비회원 비밀번호 검증 중 어떤 삭제 조건도 만족하지 못한 경우 |

## Member

| 메시지 | 발생 상황 |
|---|---|
| `존재하지 않는 회원입니다.` | 로그인 아이디, 닉네임, 회원 ID로 회원을 조회하거나 회원 탈퇴 시 대상 회원을 찾지 못한 경우 |
| `비밀번호가 올바르지 않습니다.` | 회원 탈퇴 시 입력한 비밀번호가 기존 비밀번호와 일치하지 않는 경우 |
| `아이디는 영문과 숫자만 사용해서 5~20자로 입력해야 합니다.` | 회원가입 시 로그인 아이디 형식이 올바르지 않은 경우 |
| `비밀번호는 영문과 숫자만 사용해서 6~20자로 입력해야 합니다.` | 회원가입 시 비밀번호 형식이 올바르지 않은 경우 |
| `이미 사용 중인 아이디입니다.` | 회원가입 시 이미 존재하는 로그인 아이디를 입력한 경우 |
| `이미 사용 중인 닉네임입니다.` | 회원가입 시 이미 존재하는 닉네임을 입력한 경우 |
| `닉네임은 한글, 영문, 숫자, _만 사용해서 2~12자로 입력해야 합니다.` | 회원가입 시 닉네임 형식이 올바르지 않은 경우 |

## Image

| 메시지 | 발생 상황 |
|---|---|
| `이미지가 존재하지 않습니다.` | 이미지 업로드 요청 파일이 없거나 비어 있는 경우 |
| `이미지 파일만 업로드할 수 있습니다.` | 파일 확장자 또는 실제 MIME 타입이 허용된 이미지 형식이 아닌 경우 |
| `이미지 업로드 식별값이 없습니다.` | 비회원 이미지 업로드 시 임시 이미지 소유자를 식별할 `draftToken`이 없는 경우 |
| `존재하지 않는 이미지가 포함되어 있습니다.` | 게시글 작성/수정 시 요청한 이미지 ID 중 실제 이미지 데이터가 없는 ID가 포함된 경우 |
| `이미지 매핑 권한이 없습니다.` | 게시글 작성/수정 시 현재 사용자에게 속하지 않은 임시 이미지 또는 이미 게시글에 연결된 이미지를 매핑하려는 경우 |
| `잘못된 이미지 경로입니다.` | 이미지 삭제 요청 경로가 `/images/post/`로 시작하지 않는 경우 |
| `잘못된 파일 경로입니다.` | 이미지 삭제 대상 파일 경로가 업로드 디렉터리 밖으로 벗어나는 경우 |
| `이미지 파일 삭제에 실패했습니다.` | 이미지 파일 삭제 중 `IOException`이 발생한 경우 |
| `파일 확장자가 없습니다.` | 업로드 파일명이 없거나 확장자를 포함하지 않는 경우 |

## Auth

| 메시지 | 발생 상황 |
|---|---|
| `""` | 로그인 시 입력한 아이디에 해당하는 회원을 찾지 못해 `UsernameNotFoundException`이 발생한 경우 |

## Admin

| 메시지 | 발생 상황 |
|---|---|
| `검색어를 입력해주세요.` | 관리자 회원 검색 시 검색어가 비어 있는 경우 |
| `검색 조건을 선택해주세요.` | 관리자 회원 검색 시 검색 조건이 비어 있는 경우 |
| `해당 아이디를 가진 유저가 존재하지 않습니다.` | 관리자 회원 검색에서 로그인 아이디로 조회한 회원이 없는 경우 |
| `해당 닉네임을 가진 유저가 존재하지 않습니다.` | 관리자 회원 검색에서 닉네임으로 조회한 회원이 없는 경우 |
| `올바르지 않은 검색 조건입니다.` | 관리자 회원 검색 조건이 `loginId`, `nickname` 둘 다 아닌 경우 |
